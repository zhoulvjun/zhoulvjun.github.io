周旅军   更新日期:2016-02-20

问：怎样在hexo驱动的博客中插入优酷视频？

答：

安装hexo-tag-youku插件

该插件由周旅军开发，已收录于hexo官方插件页。

在博客站点目录下执行npm install hexo-tag-youku --save，成功后会在子目录node_modules下生成hexo-tag-youku目录。
